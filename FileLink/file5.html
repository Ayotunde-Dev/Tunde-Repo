What is Web Development?

Web development is the work involved in developing a website for the internet (World Wide Web) or Intranet (Private Network). Web development can range from developing a simple static page to a more dynamic page.

Origin Of Web Development

The journey of web development began with simple HTML pages in the early days of the internet. Overtime, CSS and Javascript was introduced. CSS was developed for styling and Javascript for interactivity. This evolution transformed static websites to more dynamic and responsive webpages.

Evolution of Web Development Technologies

  • Static HTML (1990s)
  • Intriduction of CSS (late 1990s)
  • javascript(early 2000s)
  • Mobile web(2000-2010s)
  • Single Page Applications and Front End Frameworks(2010s)
  • Server side Javascript (2010s)
  • API driven development(2010 to present
  • AI and machine learning (2010 to present
